Introduction

This Project was started as a solution to the problem that data scientists spend majority of their time cleaning textual and numeric datasets before they are fit for use. We at B.A.G have tried to change this, create a tool for everyone to use for their various needs.

With this tool you'll be able to:

-Generate an error report for the dataset

-Which checks the spellings

-Fill missing values with median imputation

-Remove the duplicate row entries

-Highlight the errors in the uploaded excel file

Come give this tool and go, and find yourself making way more time for yourself!
